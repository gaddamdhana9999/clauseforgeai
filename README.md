# ClauseForge AI

AI-Powered Contract Intelligence & Negotiation Platform

## Problem

Contract review is slow, expensive, and often requires legal experts. Businesses struggle to identify risks and negotiate better contract terms.

## Solution

ClauseForge AI uses a multi-agent workflow to analyze contracts, identify risks, generate negotiation-ready redlines, compare versions, and create executive reports.

## Features

* Contract Upload
* Executive Summary
* Risk Analysis
* AI Negotiation Assistant
* Contract Comparison
* Executive Report Generation
* AI Chat Assistant

## Workflow

Upload Contract
→ Clause Extraction
→ Risk Analysis
→ Negotiation Suggestions
→ Executive Report
